Project Overview: The Challenge

The goal of this Final Year Project was to develop a polished, ready-to-publish 3D mobile game that demonstrated proficiency in game engine mechanics, efficient code architecture, and optimized mobile performance. We chose the popular helix tower concept, aiming to add unique level-generation algorithms and responsive touch controls.

Jumpy Jump is an endless arcade game where players guide a bouncing ball down a helix tower, breaking through platforms while avoiding colored obstacles. The complexity increases with procedurally generated levels and escalating speed, testing the limits of our real-time rendering and resource management skills.
